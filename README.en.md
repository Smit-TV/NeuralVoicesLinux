# NeuralVoices for Linux

This project allows you to use Microsoft Neural Voices on Linux.

## Dependencies

1. Gradle
2. Kotlin
3. Java
4. Any Microsoft Neural Voice

## Build

Run the following commands:

```bash
git clone https://github.com/Smit-TV/neuralVoices
cd neuralVoices
./build.sh
````

## Installation

After the build is complete, run:

```bash
./install.sh
```

After installation, you can get usage help with:

```bash
neuralvoices --help
```
